# RTC Insight Engine

An end-to-end Data Analytics, Data Engineering, and AI project built using Rewriting the Code (RTC) annual reports.

## Features

- PDF data extraction
- Data cleaning and normalization
- MySQL database design
- SQL analytics
- Business KPI dataset
- Power BI dashboard
- RAG chatbot (coming soon)

## Dataset

Reports analyzed:

- RTC Impact Report 2023
- RTC Impact Report 2024
- RTC Impact Report 2025
- RTC Internship Report 2024
- RTC Recruiting Report 2025

## Tech Stack

- Python
- Pandas
- MySQL
- SQL
- Power BI
- Sentence Transformers
- FAISS
- FastAPI
